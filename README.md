A rule-based exam date sheet generator built using Python and Streamlit.

## Features
- Generates exam date sheets automatically
- Enforces rule: **No three consecutive classes have the same exam on the same day**
- Supports class grouping (11 & 12 Science / Commerce)
- Holiday & Sunday exclusion
- Excel upload & download

## Rule Disclaimer
> The date sheet generation is strictly based on the rule of  
> **NO three consecutive classes having the same exam on the same day**.  
>  
> If any adjustment is required beyond this rule, it should be done **manually**.

## Tech Stack
- Python
- Streamlit
- Pandas
